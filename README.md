## Преамбула

Это проект с хакатона, проект для охранных организаций, которые хотят быть уверенными в том, что охранник совершает обходы. Для этого на объекте размещены nfc метки, которые должен сканировать охранник во время обхода.

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
